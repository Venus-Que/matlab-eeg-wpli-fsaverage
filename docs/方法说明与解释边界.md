# 方法说明与解释边界

## 1. 分析对象

每个输出矩阵属于一个被试、一个阶段、一个条件和一个频段。代码不会把不同被试的原始试次合并计算连接。

默认条件为O1-O6和S1-S6。两类事件分别执行分段、试次剔除、固定数量抽样、频谱估计和wPLI计算，因此同一阶段会得到独立的`_O_`与`_S_`结果。这样可以保留条件差异，并支持后续同一被试内的O-S配对分析；不得把O、S试次先混合后计算一个连接矩阵。

部分旧phase2记录只有P事件而没有O/S。此类记录保留为单独P条件结果，不得伪装成O或S，也不能进入O-S配对差异分析。

## 2. 事件和时间窗

默认截取目标事件后0.5-2.5秒。事件优先来自FIF；若FieldTrip无法读取MNE annotations，则读取相邻`markers.csv`并将秒转换为一基采样点。

## 3. 质量控制

每个候选试次计算所有通道峰峰值。任一通道超过500微伏时剔除整试次。若剩余试次少于6个，该记录失败，代码不会补零或降低单个被试门槛。

500微伏不是通用最佳阈值。迁移到其他设备、参考方式或预处理流程时，应依据数据单位、噪声水平和人工复核重新确定。

## 4. 统一试次数

当前固定选择6个合格试次，用于控制不同记录的样本量并复现项目预实验。固定随机种子保证同一文件重复运行得到相同选择。

六试次对连接估计通常较少。正式研究必须报告稳定性检查，例如：

- 分半矩阵相关；
- 不同随机重抽样的一致性；
- 边、节点强度和图指标的置信区间；
- 对试次数增加的敏感性分析。

## 5. 减去诱发平均

代码从每个试次中减去跨试次平均波形，以降低共同事件锁定响应被误当作试次间功能连接的风险。该操作改变了分析问题：结果更侧重非相位锁定活动。是否保留该步骤应在方案中预先声明。

## 6. 频谱与连接

代码使用FieldTrip：

```matlab
cfg.method = 'mtmfft';
cfg.output = 'fourier';
cfg.taper = 'dpss';
cfg.foilim = [1 30];
cfg.tapsmofrq = 1.0;
cfg.keeptrials = 'yes';
cfg.keeptapers = 'yes';
```

随后使用：

```matlab
cfg.method = 'wpli_debiased';
```

这是去偏平方wPLI，不等同于普通wPLI。少样本时估计可能为负；负值不是负连接。矩阵保存负值，网络图只显示正连接。

当前输出四个频段：delta 1-4 Hz、theta 4-8 Hz、alpha 8-13 Hz、beta 13-30 Hz。频点按左闭右开分配，beta额外包含30 Hz，即`[1,4)`、`[4,8)`、`[8,13)`和`[13,30]`。这样4、8、13 Hz不会在两个频段重复平均。由于DPSS设置为1 Hz平滑半带宽，边界附近的频谱信息仍可能跨越名义边界，这与重复使用同一个频点是两回事。

四频段是预先定义的描述性分析范围，不代表四个频段都会产生显著组间差异。正式比较应对频段、阶段和图指标形成的多重检验进行校正，并报告未显著结果。

## 7. fsaverage图的含义

代码用FieldTrip `standard_1005.elc`确定传感器方向，再将节点投影到fsaverage pial表面。其用途是提供一致的三维背景和网络展示。

正确表述：

> 传感器空间wPLI网络在fsaverage皮层表面上的投影可视化。

错误表述：

> fsaverage皮层脑区之间的源空间连接。

真正的源空间连接还需要个体或模板前向模型、源重建、脑区分区、源时间序列提取以及源泄漏控制。

## 8. 跨组比较

如果不同诊断组使用了不同电极布局，布局本身可能造成组间差异。正式比较前至少应：

1. 审核每组的电极数量和名称分布；
2. 限定相同布局，或统一到所有记录共有节点；
3. 不用ECG/EOG填充缺失EEG节点；
4. 不直接对插值产生的连接作生物学解释；
5. 在统计模型中控制年龄、性别、教育等预先定义的混杂因素。

本流程允许59导记录生成个体图，是为了保留可审查结果，不代表59×59矩阵可以与64×64矩阵直接合并。59导图片标题和批处理状态会明确标注这一限制。

### 本项目导数审计结果

- 64导记录：19条，内部使用同一套64节点标签。
- 59导记录：12条，内部使用同一套59节点标签。
- 两套标签的公共节点：54个。

59导不是从当前64导列表中简单缺少5个节点，两套记录采用了不同的电极组合。代码不改变原始导数：59导按59×59计算，64导按64×64计算；不会补零、插值或制造缺失节点。

如果后续必须联合分析，可预先定义54个公共通道，并从每条原始FIF重新提取这54个通道后计算54×54矩阵。“54导”只是一种统一节点的二次分析方案，不是当前个体图的计算导数。

## 9. 诊断用途

该仓库不提供临床诊断工具。任何AD、SCD或其他诊断分类必须在独立样本、严格交叉验证和预注册特征流程下验证。
